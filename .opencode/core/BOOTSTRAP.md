# Bootstrap

## Objetivo
Inicializar automáticamente cualquier proyecto, detectando si es nuevo o existente, y preparando la estructura IA necesaria.

## Pipeline
Discovery → Knowledge → Memory → Context Compression → Stack Detection →
Domain Detection → Agent Factory → Skill Factory → Planning →
Approval → Implementation → Validation → Audit → Memory Update

## Flujo detallado
1. Detectar si el proyecto es nuevo o existente
2. Ejecutar Discovery (stack, dominio, arquitectura, dependencias, riesgos)
3. Generar PROJECT_PROFILE con todo el contexto detectado
4. Generar PROJECT_MEMORY con historial inicial
5. Detectar stack tecnológico (lenguajes, frameworks, BD, cloud)
6. Detectar dominio de negocio
7. Detectar arquitectura (MVC, Clean, Microservicios, Monolito Modular)
8. Crear agentes dinámicos necesarios (solo los que aplican al stack/dominio)
9. Crear skills dinámicas necesarias
10. Generar plan de trabajo inicial
11. Esperar aprobación del usuario
12. Iniciar implementación

## Outputs generados
- PROJECT_PROFILE
- PROJECT_MEMORY
- Agentes relevantes al proyecto
- Skills relevantes al proyecto
- Roadmap inicial
- Checklists
- Templates necesarios
